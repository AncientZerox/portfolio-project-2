About the Project

This project focuses on developing a gas leak detector using various sensors. The device is designed to monitor the presence of flamable gases in the environment and alert users in case of a leak. It aims to enhance safety in homes and workplaces by providing real-time gas monitoring.

Features

  • Real-time gas detection
  • Audible and visual alarms
  • Compact and portable design
  • Easy installation and setup
